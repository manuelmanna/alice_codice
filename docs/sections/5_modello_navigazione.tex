\section{Modello di Navigazione}

\subsection{Struttura delle pagine}
Il sistema A.L.I.C.E. prevede due viste completamente separate: una per l'operatore sanitario e una per il paziente anziano. L'accesso avviene attraverso una pagina di login comune, dalla quale l'utente viene indirizzato alla vista appropriata in base al proprio ruolo.

\subsection{Site Map -- Vista Operatore Sanitario}

\begin{itemize}
    \item \textbf{Dashboard:} panoramica con widget informativi (pazienti attivi, alert, statistiche di aderenza, mood check del giorno).
    \item \textbf{Gestione Pazienti:} CRUD completo con ricerca, filtri, dettaglio paziente e contatti di emergenza.
    \item \textbf{Attivit\`a:} pianificazione terapeutica con selezione esercizi da un catalogo.
    \item \textbf{Monitoraggio:} grafici e tabelle con risultati Mood Check, punteggi cognitivi e aderenza.
    \item \textbf{Profilo:} gestione del proprio account operatore.
\end{itemize}

\begin{itemize}
    \item \textbf{Login Semplificato:} accesso con codice numerico, senza password complesse.
    \item \textbf{Home / Piano Giornaliero:} card grandi e colorate con le attività del giorno e il loro stato.
    \item \textbf{Esercizi Fisici:} lista esercizi con istruzioni semplificate.
    \item \textbf{Giochi Cognitivi:} accesso diretto al Memory Card, con carte grandi e feedback gratificanti.
    \item \textbf{Mood Check:} emoji grandi per comunicare lo stato d'animo.
    \item \textbf{Farmaci:} elenco farmaci del giorno con pulsante di conferma per ciascuno.
\end{itemize}

Principi di navigazione: massimo 2 livelli di profondità, pulsante ``Torna indietro'' sempre visibile, nessun menu hamburger, feedback visivo immediato per ogni azione.
